# ADR: 0088-event-sourcing-for-audit
**Date**: 2021-11-05
**Status**: Accepted

## Context
Compliance requires strict audit logs. We are shifting to Event Sourcing for the core task aggregate.

## Decision
We will implement this immediately to support our scaling needs.

## Consequences
Significant refactoring required, but it positions us for the next 5 years of growth.
